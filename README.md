## Hi, I'm Sadan! 👋

🚀 Co-founder of **MacroRanked** — an AI-powered fitness platform using computer vision to estimate calories from food images, built with React Native, FastAPI and Supabase. Launching Q4 2026.<br>
🫀 Conducting **ML research in cardiac electrophysiology** at Warwick Medical School — developing models to classify optical mapping waveforms into healthy and pathological cardiac tissue.<br>
🧠 Building **Atlas** — a modular personal AI operating system featuring multiple specialised LLM agents, semantic memory (ChromaDB), retrieval-augmented generation (LlamaIndex), and local Llama 3.1 inference.<br>
🍽️ Researching **computer vision for food recognition** using transfer learning and ResNet-50, including a full ablation study across 9 training recipes on Food-101 (101 classes, 75,750 images).<br>
⚽ Built a **live World Cup 2026 ML prediction engine** using historical match data and real-time tournament results to forecast knockout-stage outcomes.<br>
🎓 BSc Health & Medical Sciences @ University of Warwick<br>
📚 Currently studying: Deep Learning • Computer Vision • Andrew Ng ML Specialization<br>
💡 Interests: medical AI • cardiac EP • computer vision • ML research • full-stack development
